Abstract

Existing weakly supervised semantic segmentation methods in computational pathology rely on a multi-stage paradigm: class activation map generation, offline pseudo-mask refinement, and fully supervised retraining. While established, this decoupled approach incurs high computational training costs and suffers from error propagation, where local texture biases in shallow CNN layers generate false-positive artifacts that subsequent refinement steps often fail to correct. We propose Single-Stage Hierarchical Rectification (SSHR), a framework that proactively purifies intermediate feature representations during the forward pass. SSHR introduces a Hierarchical Feature Rectification Module that uses deep global semantic context to filter local anomalies in shallow layers, generating high-fidelity activation maps directly within a single training loop. Experiments on LUAD-HistoSeg and BCSS show that SSHR outperforms state-of-the-art multi-stage methods while reducing training duration by 2 to 5 times.

Method

SSHR internalizes CAM purification instead of correcting noisy masks after generation.

Global Semantic Rectification

Deep features encode global tissue context. SSHR projects this context into channel-wise gates that attenuate low-level feature channels likely to activate absent or implausible tissue classes.

Contextual Homogenization

A lightweight large-kernel depthwise convolution suppresses isolated spatial outliers by aligning each location with its broader tissue neighborhood.

Stable Residual Integration

Zero-initialized learnable residual scales let the model start as the pretrained backbone and gradually learn how strongly to apply semantic and spatial rectification.

Efficient by Design

By bypassing pseudo-mask generation and segmentation retraining, SSHR reduces both training time and inference latency.

No offline pseudo-mask stage

CAM refinement happens inside the model, removing expensive post-processing loops.

No second-stage retraining

SSHR avoids the unstable dependency on a separate segmentation backbone trained on noisy pseudo labels.

Fast inference

The reported latency is 9.10 ms per patch, faster than all compared multi-stage competitors.
